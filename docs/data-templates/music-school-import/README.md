# 琴行约课真实数据导入模板说明

当前推荐只填写一个 Excel 文件：

- `music-school-import-template.xlsx`

这个文件包含 4 个工作表：

- `校区信息`：填写一店、二店等校区资料。
- `老师信息`：填写老师资料。
- `学生信息`：填写学生、课程、老师、课时和缴费状态。
- `填写说明`：解释每个字段怎么填。

## 填写原则

- 尽量使用中文填写，方便第一次整理资料的人直接理解。
- 不要求填写系统编号、合同内部编号、绑定编号等技术字段，后续导入脚本可以自动生成。
- 日期统一用 `YYYY-MM-DD`，例如 `2026-06-11`。
- 不确定的信息可以先留空，后续需要时再补。
- 一个学生如果有多门课程，就在 `学生信息` 工作表里填多行。

## 校区信息

| 字段 | 是否必填 | 怎么填 |
| --- | --- | --- |
| 校区名称 | 必填 | 完整名称，例如 `菠菜现代音乐一店（净月御翠园）`。 |
| 校区简称 | 必填 | 建议填 `一店`、`二店`。老师和学生表用这个名称关联校区。 |
| 校区地址 | 必填 | 详细地址。 |
| 联系电话 | 可选 | 暂未统一可填 `请联系校区老师`。 |
| 营业时间 | 可选 | 例如 `周二至周日 12:00-20:00`。 |
| 备注 | 可选 | 其他说明。 |

## 老师信息

| 字段 | 是否必填 | 怎么填 |
| --- | --- | --- |
| 所属校区 | 必填 | 填 `一店` 或 `二店`。 |
| 老师姓名 | 必填 | 例如 `林老师`。 |
| 手机号 | 可选 | 老师联系电话。 |
| 主授课程 | 必填 | 例如 `架子鼓`、`吉他`、`钢琴`。 |
| 可授课程 | 可选 | 多个课程可填 `架子鼓、吉他`。 |
| 在职状态 | 必填 | 建议填 `在职` 或 `停用`。 |
| 备注 | 可选 | 排班习惯、特殊说明等。 |

## 学生信息

学生信息已经精简为第一批导入最需要的字段。

| 字段 | 是否必填 | 怎么填 |
| --- | --- | --- |
| 所属校区 | 必填 | 填 `一店` 或 `二店`。 |
| 学生姓名 | 必填 | 学生姓名。 |
| 联系电话 | 可选 | 家长或学生联系电话。 |
| 课程名称 | 必填 | 例如 `架子鼓`、`吉他`、`钢琴`。 |
| 授课老师 | 必填 | 填老师姓名，例如 `林老师`。 |
| 课程类型 | 必填 | 建议填 `固定课时` 或 `按册学习`。 |
| 按册级别 | 按需填写 | 课程类型为 `按册学习` 时填写，例如 `第一册`。 |
| 总课时 | 按需填写 | 课程类型为 `固定课时` 时填写，例如 `20`。 |
| 已上课时 | 必填 | 已经消耗的课时；没有历史可填 `0`。 |
| 报名日期 | 可选 | 格式 `YYYY-MM-DD`。 |
| 到期日期 | 可选 | 格式 `YYYY-MM-DD`。 |
| 缴费状态 | 必填 | 建议填 `已付清`、`分期`、`体验课`。 |
| 在读状态 | 必填 | 建议填 `在读`、`停用`、`已到期`、`体验课`。 |
| 当前进度 | 可选 | 当前学习进度。 |
| 备注 | 可选 | 特殊提醒、学习情况、换老师历史等。 |

## 第一批整理建议

先把下面这些信息填准：

- 学生在哪个校区。
- 学生学什么课程。
- 由哪位老师授课。
- 是固定课时还是按册学习。
- 固定课时还剩多少，也就是 `总课时 - 已上课时`。
- 是否分期、是否在读。

更细的合同附件、历史预约、历史上课记录、财务结算等信息，后续需要时再补。
