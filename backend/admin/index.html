真实数据导入

正式导入仍建议在服务器命令行执行，导入脚本会先生成备份，避免 Excel 误填后直接覆盖线上数据。

cd /home/ubuntu/music-school-booking/backend
DATABASE_PATH=/var/lib/spinach-music/spinach-music.json npm run import:music-school -- --replace
sudo systemctl restart spinach-music-api

巡检账号

小程序前端不设管理员入口。使用同一个手机号 13800000010，可分别进入学员端和老师端做上线巡检。

备份当前数据库

生成服务器本地 JSON 备份，适合每次批量导入、批量修改前执行。

导出当前数据

下载当前数据库快照，便于本地留档、人工检查或发给开发继续排查。

恢复数据

恢复会覆盖正式库，暂不放浏览器按钮。需要恢复时请在服务器执行：

cd /home/ubuntu/music-school-booking/backend
DATABASE_PATH=/var/lib/spinach-music/spinach-music.json npm run restore -- /var/lib/spinach-music/backups/备份文件.json
sudo systemctl restart spinach-music-api
